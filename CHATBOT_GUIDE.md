## 시작하기

카카오톡에서 **바이오푸드랩** 채널을 추가하고 채팅을 시작하세요.

```
안녕하세요! 바이오푸드랩 챗봇 [바푸]입니다.
원하시는 서비스를 선택해주세요.

[검사분야] [검사주기] [검사항목]
```

---

## 메인 메뉴

| 버튼 | 설명 |
|------|------|
| **검사분야** | 다양한 검사 서비스 안내 (자가품질검사, 영양성분검사 등) |
| **검사주기** | 식품유형별 검사 주기 안내 |
| **검사항목** | 식품유형별 검사 항목 안내 |
| **처음으로** | 메인 메뉴로 돌아가기 |
| **이전** | 이전 단계로 돌아가기 |

## 검사분야

### 접근 방법
```
메인 메뉴 → [검사분야]
```

### 제공 서비스
- **자가품질검사** - 식품/축산 자가품질검사 안내
- **영양성분검사** - 5대/9대/14대 영양성분 검사
- **항생물질** - 동물용 의약품 안전관리
- **잔류농약** - 농산물 안전성 검사
- **방사능** - 방사능 검사 (HPGe)
- **비건/할랄/동물DNA** - 비건, 할랄, 동물 DNA 검사
- **알레르기** - Allergen 분석 (ELISA/RT-PCR)
- **글루텐Free** - 글루텐 프리 검사
- **소비기한설정** - 소비기한 설정 실험
- **이물질검사** - 이물질 분석 (XRF, FT-IR)

## 검사주기

### 사용 방법
```
[검사주기] → 분야 선택 (식품/축산) → 업종 선택 → 식품유형 입력
```

### 예시
```
사용자: 검사주기
챗봇: 분야를 선택해주세요.
      [식품] [축산]

사용자: 식품
챗봇: 업종을 선택해주세요.
      [식품제조가공업] [즉석판매제조가공업]

사용자: 식품제조가공업
챗봇: 식품유형을 입력해주세요. (예: 과자, 빵류, 음료류)

사용자: 과자
챗봇: 📋 과자의 검사주기
      • 검사주기: 3월 1회 이상
      ...
```

## 비용 문의

### 키워드
다음 키워드가 포함된 질문 시 비용 안내로 연결됩니다:
- 비용, 수수료, 가격, 단가, 얼마, 요금

### 사용 방법
```
사용자: 검사 비용이 얼마예요?
챗봇: 💰 검사 비용 안내입니다.
      어떤 검사의 비용을 확인하시겠어요?
      [영양성분] [자가품질검사] [기타] [처음으로]
```

### 영양성분 비용
```
사용자: 영양성분 비용
챗봇: 💰 영양성분 검사 비용 안내입니다.
      [5대 영양성분] [9대 영양성분] [14대 영양성분]

사용자: 5대 영양성분
챗봇: 💰 5대 영양성분 검사비용
      • 종류: 어린이기호식품 5대
      • 항목: 열량, 당류, 단백질, 포화지방, 나트륨
      • 수수료: 150,000원 (부가세 별도)
      • 소요기간: 영업일 기준 12일
      [이전] [처음으로]
```

## 자연어 질문

### 개요
정해진 버튼 외에도 **자연어로 질문**하면 관련 Q&A를 검색해서 안내합니다.

### 사용 예시
```
사용자: 소비기한을 늘리고 싶어요
챗봇: 🔍 관련 Q&A를 찾았습니다:
      1. Q1. 소비기한설정실험이란?
      2. Q2. 소비기한설정실험 종류
      3. Q3. 소비기한설정실험이 생략 가능한 경우
      번호를 선택해주세요.
      [1] [2] [3] [더보기] [처음으로]
```

### 답변 확인
```
사용자: 1
챗봇: 📌 [소비기한] Q1. 소비기한설정실험이란?

      소비기한 제품의 품질을 고려하여 보장할 수 있는...
      (상세 답변 내용)

      💡 원하시는 답변이 아니라면 [이전]을 눌러주세요. (남은 횟수: 3회)
      [이전] [처음으로]
```

### 더보기 기능
검색 결과가 5개 이상일 경우 **[더보기]** 버튼으로 추가 결과를 확인할 수 있습니다.

## 이미지 인식 (OCR)

### 개요
**품목제조보고서** 또는 **영업등록증** 이미지를 업로드하면 자동으로 식품유형을 인식합니다.

### 사용 방법
1. [검사주기] 또는 [검사항목] 선택
2. 분야 및 업종 선택
3. 식품유형 입력 대신 **이미지 업로드**

### 인식 가능한 표기
- 식품유형
- 식품의 유형
- 식품의 종류
- 품목유형
- 제품유형

### 예시
```
챗봇: 식품유형을 입력해주세요.
사용자: (품목제조보고서 이미지 업로드)
챗봇: 📷 이미지에서 '과자'을(를) 찾았습니다.
      📋 과자의 검사주기...
```

## 함량 계산

### 접근 방법
```
메인 메뉴 → [검사분야] → [영양성분검사] → [함량계산]
```

### 제공 기능

#### 1. 배합 함량 계산
제품 총 중량과 원재료 중량을 입력하면 배합비율(%)을 계산합니다.

```
사용자: 배합 함량
챗봇: 제품의 총 중량(g)을 입력해주세요.

사용자: 1000
챗봇: 원재료명과 중량을 입력해주세요.
      예) 설탕 200, 밀가루 300, 버터 150

사용자: 설탕 200, 밀가루 300, 버터 150
챗봇: 📊 배합함량 계산 결과 (총 중량: 1000g)

      1. 밀가루: 300g → 30.00%
      2. 설탕: 200g → 20.00%
      3. 버터: 150g → 15.00%
      ─────────────────
      합계: 650g (65.00%)
```

#### 2. 당알코올 계산
당알코올의 열량 기여도를 계산합니다.

```
사용자: 당알코올 계산
챗봇: 제품의 총 중량(g)을 입력해주세요.

사용자: 100
챗봇: 당알코올의 배합 함량(g)을 입력해주세요.

사용자: 10
챗봇: 당알코올 순 함량(%)을 입력해주세요.
      ※ 한글표시사항 또는 성적서에서 확인 가능

사용자: 98
챗봇: 수분값을 적용하시겠습니까?
      [수분값 적용] [수분값 미적용]

사용자: 수분값 미적용
챗봇: 📊 당알코올 계산 결과
      • 총 중량: 100g
      • 당알코올 함량: 10g
      • 순 함량: 98%
      • 당알코올 열량: 약 23.52 kcal
```

## 자주 묻는 질문

### Q. 버튼이 안 보여요
A. 채팅 입력창 옆의 **+** 버튼을 눌러 버튼 메뉴를 확인하세요.

### Q. 이전 단계로 돌아가고 싶어요
A. **[이전]** 버튼을 누르거나 "이전"이라고 입력하세요.

### Q. 처음부터 다시 시작하고 싶어요
A. **[처음으로]** 버튼을 누르거나 "처음으로" 또는 "종료"라고 입력하세요.

### Q. 원하는 답변이 없어요
A. 자연어로 질문해보세요. 예: "알레르기 검사 방법이 뭐예요?"
